# Simple-ATM-Simulation
A simple ATM simulation system written in C++ featuring PIN authentication, account balance management, and basic transaction operations with console-based interaction.

# 🏦 ATM Simulator (C++)

A simple console-based ATM simulation system built in C++ for educational purposes.

---

# 📌 Features

- 🔐 PIN authentication system
- 💰 Check account balance
- 💸 Withdraw cash
- 💳 Deposit cash
- 🧠 Input validation system
- ⏳ Basic lock delay after failed attempts

---

# 🏗 Architecture

User Input
↓
PIN Check System
↓
ATM Menu System
↓
Transaction Handler
↓
Balance Update


---

# ▶️ How to Run

## Compile


g++ main.cpp -o atm
